# Plant System Skeleton

Project skeleton for a smart plant monitoring and care support system using IoT and AI.

## Goal

- Only create the basic folder structure and starter files.
- Do not complete the implementation.
- Each team member can continue from the comments and TODO notes in each module.

## Suggested Structure

- `backend/`: API server, rule engine, and database layer.
- `ai-service/`: plant recognition service from images.
- `dashboard/`: web interface.
- `iot/esp32/`: ESP32 firmware.
- `docs/`: API notes and team split.

## Approved APIs

See `docs/API_SPEC.md` for details.

## Notes

- Backend and dashboard are scaffolded with a JavaScript-oriented structure for easier team continuation.
- The AI service uses a Python stub because it fits the model inference workflow.
- The ESP32 module uses a C++ stub.
